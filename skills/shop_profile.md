# Project Skills: お酒とおばんざい つづみ

## 1. ブランドアイデンティティ (Brand Identity)
- **店名**: お酒とおばんざい つづみ
- **ジャンル**: 和食・小料理・おばんざい
- **コンセプト**: 旬の食材を活かした丁寧なおばんざいと、店主厳選の日本酒。
- **デザインキーワード**: 木のぬくもり, 隠れ家, 清潔感, 縦書き, 墨文字, 落ち着いた朱色

## 2. 視覚要素とアニメーション設定 (Antigravity Directives)
- **全体演出**: 鼓の音のように、静かだが芯のあるリズム感。
- **背景**: 質感のある和紙（生成り色）をベースに。
- **エフェクト**: スクロールに合わせて、料理の写真が「水面に浮かぶ木の葉」のように滑らかにフェードイン。
- **フォント**: 見出しは高級感のある明朝体（Noto Serif JP等）、本文は読みやすいゴシック体。
